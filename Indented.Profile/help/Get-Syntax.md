# Get-Syntax

## SYNOPSIS
Get the syntax for a command.

## SYNTAX

### ByName
```
Get-Syntax [-Name] <String> [-Short] [<CommonParameters>]
```

### FromCommandInfo
```
Get-Syntax -CommandInfo <CommandInfo> [-Short] [<CommonParameters>]
```

## DESCRIPTION
Get the syntax for a command.
A wrapper for Get-Command -Syntax.
